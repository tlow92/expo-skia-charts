## Time Series

### Daily Data

Display time-based data with formatted dates:

<ChartDemo
  title="Daily Time Series"
  config={{
    data: Array.from({ length: 30 }, (_, i) => ({
      x: new Date(2024, 0, i + 1).getTime(),
      y: 50 + Math.sin(i / 5) * 30 + Math.random() * 10
    })),
    xAxis: {
      enabled: true,
      isTimeData: true,
      showGridLines: true,
      formatter: (value) =>
        new Intl.DateTimeFormat('en-US', {
          month: 'short',
          day: 'numeric',
        }).format(new Date(value)),
    },
    yAxis: {
      enabled: true,
      showGridLines: true,
    }
  }}
  height={300}
/>

## Interactive Charts

### Full Interactive Chart

Chart with all interactive features enabled:

<ChartDemo
  title="Fully Interactive"
  config={{
    data: Array.from({ length: 30 }, (_, i) => ({
      x: new Date(2024, 0, i + 1).getTime(),
      y: 50 + Math.sin(i / 5) * 25 + Math.random() * 10
    })),
    hover: {
      enabled: true,
      showDot: true,
      highlightLine: true,
      tooltip: {
        renderContent: (dataPoint) => (
          <div style={{
            backgroundColor: 'rgba(0, 0, 0, 0.8)',
            color: 'white',
            padding: '8px 12px',
            borderRadius: '4px',
            fontSize: '14px',
          }}>
            <div>{new Date(dataPoint.x).toLocaleDateString('en-US')}</div>
            <div style={{ fontWeight: 'bold', marginTop: '4px' }}>
              {dataPoint.y.toFixed(1)}
            </div>
          </div>
        ),
        snapToPoint: true,
      },
    },
    xAxis: {
      enabled: true,
      isTimeData: true,
      showGridLines: true,
    },
    yAxis: {
      enabled: true,
      showGridLines: true,
    }
  }}
  height={300}
/>

## Real-World Use Cases

### Stock Price

Simulated stock price chart:

<ChartDemo
  title="Stock Price"
  config={{
    data: Array.from({ length: 60 }, (_, i) => {
      const basePrice = 150;
      const trend = i * 0.5;
      const volatility = Math.random() * 10 - 5;
      return {
        x: new Date(2024, 0, 1 + i).getTime(),
        y: basePrice + trend + volatility
      };
    }),
    hover: {
      enabled: true,
      showDot: true,
      highlightLine: true,
      tooltip: {
        renderContent: (dataPoint) => (
          <div style={{
            backgroundColor: 'white',
            padding: '8px 12px',
            borderRadius: '4px',
            border: '1px solid #e0e0e0',
            boxShadow: '0 2px 8px rgba(0,0,0,0.1)',
          }}>
            <div style={{ fontSize: '12px', color: '#666' }}>
              {new Date(dataPoint.x).toLocaleDateString('en-US', {
                month: 'short',
                day: 'numeric',
                year: 'numeric'
              })}
            </div>
            <div style={{ fontSize: '16px', fontWeight: 'bold', color: '#000', marginTop: '4px' }}>
              ${dataPoint.y.toFixed(2)}
            </div>
          </div>
        ),
        snapToPoint: true,
      },
    },
    colors: {
      lineBase: '#10B981',
      highlightColor: '#059669',
      dotBase: '#047857',
    },
    xAxis: {
      enabled: true,
      isTimeData: true,
      showGridLines: true,
      tickCount: 6,
    },
    yAxis: {
      enabled: true,
      showGridLines: true,
      formatter: (value) => `$${value.toFixed(0)}`,
    }
  }}
  height={350}
/>
